CalculatieTool.com signup and contact form.

== Description ==

The WordPress CTSignup Plugin is a client tot the CalculatieTool.com application. It can be used to either signup new users via OAuth2 or to use a standardized contact form. Add shortcodes to the current WordPress pages or post to create a new form. 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/CTSignup` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->CTSignup screen to configure the plugin, add the keys and test the settings.
4. If the OAuth keys are correct and the testmail is received, the plugin is ready to be used.
